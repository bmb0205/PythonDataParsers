
This file describes the usage of contained json files in regards to the user of this package setting specific output parameters. 

The user can edit the respective json file of a desired database source and specify which data attributes will be written to outfile.

This shows an genertic example of how to modify the json file to output only specified attributes/columns.

		User can place '++' as first two characters (index 0 and 1) of attribute string to designate desired columns to be exported from each file.

		Example:

		"file.info" : ['++firstAttribute,
					   'secondAttribute'
					   '++thirdAttribute',
					   'fourthAttribute'],
		"otherfile.info" : ['attribute',
							'++anotherAttribute',
							'onemoreAttribute']

		This will write only 'firstAttribute' and 'thirdAttribute' columns to .csv output for file.info, and only 'anotherAttribute' for otherfile.info.
